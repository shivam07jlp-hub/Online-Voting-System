# Online Voting System

A secure Online Voting System developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features
- Student Registration
- OTP Verification
- Student Login
- Admin Login
- Candidate Management
- One Student One Vote
- Election Result Dashboard

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Installation

```bash
npm install
npm start
```
